# Business Landing Website

This project is a simple landing page website for a business, built as part of the Udacity course.

## Project Structure

- `index.html`: The main home page.
- `css/`: Contains stylesheets for different pages.
  - `global.css`: Global styles.
  - `home.css`: Styles for the home page.
  - `contact.css`: Styles for the contact page.
- `images/`: Folder for images used in the website.
- `page/contact.html`: The contact page.

## Technologies Used

- HTML5
- CSS3

## How to View

Simply open `index.html` in any web browser.
